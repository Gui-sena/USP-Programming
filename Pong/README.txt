No terminal:
	1 - Vá para o diretório onde estão armazenados os arquivos;
	2 - $javac Score.java (compile o código do Placar);
	3 - $javac Player.java (compile o código do Jogador);
	4 - $javac Ball.java (compile o código do Ball);
	5 - $java Pong (execute o código do Jogo Pong que o jogo irá rodar logo em seguida);
